## Install libraries
 Run scripts in root folder
- cp .env.example .env
- yarn 
- yarn start

## Migrate database
- yarn db:migrate
- use https://sequelize.org/docs/v6/other-topics/migrations/

## Postgree 
- use GUI POSTGRE https://www.beekeeperstudio.io/
- change value in file .env 

## JWT 
- Go to https://jwt.io/ copy SECRET_KEY to .env
